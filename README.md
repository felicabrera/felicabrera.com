# Felicabrera.com

A personal portfolio website built with Next.js.

[![Netlify Status](https://api.netlify.com/api/v1/badges/39aa23e3-b415-4fa1-bcc3-048a01eab084/deploy-status)](https://app.netlify.com/projects/felicabrera/deploys)

## Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deployment

Deployed on Netlify.
